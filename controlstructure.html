Control Structure in PL/SQL

Control structures in PL/SQL are used to control the flow of execution of statements. These structures allow conditional execution, repetition of code blocks, and handling multiple conditions based on logic.

There are three main types of control structures in PL/SQL:

  • Conditional Control (IF Statements)
  • Iterative Control (LOOP Statements)
  • Sequential Control (GOTO Statement)

1. Conditional Control - IF Statements

Conditional control is used to execute a block of statements based on a condition.

Types of IF Statements:

  • IF...THEN
  • IF...THEN...ELSE
  • IF...THEN...ELSIF...ELSE

Example of IF...THEN

DECLARE
  v_marks NUMBER := 80;
BEGIN
  IF v_marks >= 50 THEN
    DBMS_OUTPUT.PUT_LINE('Pass');
  END IF;
END;

Example of IF...THEN...ELSE

DECLARE
  v_marks NUMBER := 40;
BEGIN
  IF v_marks >= 50 THEN
    DBMS_OUTPUT.PUT_LINE('Pass');
  ELSE
    DBMS_OUTPUT.PUT_LINE('Fail');
  END IF;
END;

Example of IF...THEN...ELSIF...ELSE

DECLARE
  v_marks NUMBER := 75;
BEGIN
  IF v_marks >= 90 THEN
    DBMS_OUTPUT.PUT_LINE('Excellent');
  ELSIF v_marks >= 75 THEN
    DBMS_OUTPUT.PUT_LINE('Very Good');
  ELSIF v_marks >= 50 THEN
    DBMS_OUTPUT.PUT_LINE('Pass');
  ELSE
    DBMS_OUTPUT.PUT_LINE('Fail');
  END IF;
END;

2. Iterative Control - LOOP Statements

PL/SQL provides three types of loops:

  • Simple LOOP
  • WHILE LOOP
  • FOR LOOP

Example of Simple LOOP

DECLARE
  i NUMBER := 1;
BEGIN
  LOOP
    DBMS_OUTPUT.PUT_LINE('Number: ' || i);
    i := i + 1;
    EXIT WHEN i > 5;
  END LOOP;
END;

Example of WHILE LOOP

DECLARE
  i NUMBER := 1;
BEGIN
  WHILE i <= 5 LOOP
    DBMS_OUTPUT.PUT_LINE('Value: ' || i);
    i := i + 1;
  END LOOP;
END;

Example of FOR LOOP

BEGIN
  FOR i IN 1..5 LOOP
    DBMS_OUTPUT.PUT_LINE('Iteration: ' || i);
  END LOOP;
END;

3. Sequential Control - GOTO Statement

The GOTO statement is used to transfer control to a labeled statement within the PL/SQL block.

Example of GOTO Statement

DECLARE
  v_num NUMBER := 5;
BEGIN
  IF v_num = 5 THEN
    GOTO skip_block;
  END IF;
DBMS_OUTPUT.PUT_LINE('This will not be printed');
<<skip_block>>
DBMS_OUTPUT.PUT_LINE('Execution jumped to here');
END;
